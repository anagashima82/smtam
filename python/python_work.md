## 9章 ワーク2: break・continue

#### 問題

#### 以下の処理を実装してください。

2. 1から20まで繰り返し、3の倍数はスキップして表示する

```python
# 1. breakの使用
num = 1
while num <= 100:
    print(num)
    # 50になったらbreakする
    num += 1

for i in range(1, 21):
    # 3の倍数はcontinueでスキップ
    print(i)
```
